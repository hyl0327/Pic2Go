# Pic2Go Client

### Environment

* Xcode 9.3 
* Swift 4 
* iOS 11.3
* ruby 2.5.0
* Alamofire 4.7

### Installing

A step by step series of examples that tell you how to get a development env running

[Alamofire source](https://github.com/Alamofire/Alamofire)

Install CocaPods

```
gem install cocoapods
```

And

```
pod install
```
